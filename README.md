# auto-docker

这个是一个一键自动化安装最新版本docker的脚本文件，拿来即用～

* 支持centos系统  
ubuntu系统支持正在路上～

安装最新版docker使用方法：

```shell
1. 拉取项目
git clone https://github.com/Stevent-fei/auto-docker.git

2. 进入项目
cd auto-docker/docker

3. 执行脚本进行自动化安装docker
bash auto-install-docker.sh
```

卸载docker使用方法：

```shell
直接执行uninstall-docker.sh脚本即可
bash auto-uninstall-docker.sh
```
